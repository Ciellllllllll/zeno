Zeno

Zeno is a small game engine project built for learning and portfolio purposes.
It uses a Rust engine core and a C++ native backend, connected through a C ABI boundary.

Goals:
- Rust engine core
- C++ DirectX 11 backend
- C++ Game SDK
- C++ GameModule DLL
- Playable sample game

Non-goals:
- Not a Unity or Unreal replacement
- No editor in the first milestone
- No multi-platform support in the first milestone
- No Vulkan or DirectX 12 in the first milestone

Recommended Development Environment

- Windows 10/11
- Visual Studio 2022 Community
- Desktop development with C++
- MSVC v143
- Windows SDK
- Rust stable toolchain
- VS Code
- rust-analyzer
- CMake Tools
